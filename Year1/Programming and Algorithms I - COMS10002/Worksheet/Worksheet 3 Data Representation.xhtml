Mike Fraser
Week 3

Do not use a calculator on any of the questions below. You must be able to do this using mental arithmetic, or a piece of paper if you require it. At the end of this worksheet you will find a table with ASCII encodings.

  1. Assume that we have the datatypes char and int.
    1. Which set does 0 belong to? Give two other example values of that datatype.
    2. Which set does '0' belong to? Give two other example values of that datatype.
    3. Which set does '\0' belong to? Give two other example values of that datatype.

  2. Assume an 8-bit machine. Write down the binary representations for:
    1. The int 0
    2. The int 1
    3. The int 10
    4. The int 100
    5. The int -1
    6. The int -10
    7. The int -100
    8. The unsigned 10
    9. The unsigned 100
    10. The unsigned 156
    11. The unsigned 246
    12. The unsigned 255
    13. The character 'd'
    14. The character '+'

  3. Assume an 8-bit machine. What is the interpretation of each of the following bit patterns as an int, unsigned, and char?
    1. 01100110
    2. 01000110

  4. Assume an 8-bit machine programmed in C. How much is
    1. int 100 + 100 + 100 + 100
    2. unsigned 100 + 100 + 100 + 100
    3. int 100 + 100 + 100 - 400
    4. unsigned 100 + 100 + 100 - 400
    5. char '1' + 2
    6. char '2' + 1
    7. char '1' + '2'
    8. char 'p' + ('A' - 'a')

  5. Assume a 32-bit floating point number, containing a sign bit s, an 8-bit unsigned biased exponent e, and a 23-bit mantissa m, such that the value of a number s|e|m is (-1)s2e-127(1 + m/223). Write down the representations for
    1. 1
    2. 0.5
    3. 0.125
    4. -257
    5. 0.1

      And interpretations for

    6. 0 10000000 10000000000000000000000
    7. 0 01111111 01010101000000000000000

      How much is (note 2-23 is 0.0000001192...)

    8. 1 + 0.1
    9. 1 + 0.0000001
    10. 1 + 0.00000005
    11. (((1 + 0.00000005) + 0.00000005) + 0.00000005)
    12. (1 + (0.00000005 + (0.00000005 + 0.00000005)))

    1. In a computer language we make identifiers out of characters. In C, an identifier consists of a letter or underscore, followed by any number of letters, digits or underscores. How many different identifiers are there with up to 7 characters? (do NOT use a caclulator, but write it down in terms of multiplications and additions, then make a guess as to how many zeroes the answer has). Note: case is important.
    2. Write down the formula for identifiers of exactly N characters long.
